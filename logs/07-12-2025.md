# Day4logs
- "/bin" is basic command and integrated with /usr/bin.
- "/sbin" is commands for system control or maintenance.
- "/usr/bin" includes most of applications.
- "/etc" includes all of setting for Linux system.
- "/var/log" is log storage. 
- "/home" is place to settle data for personal users.
- "/opt" includes applications for unrelated to package management system.
- "/dev" includes files for devices.
